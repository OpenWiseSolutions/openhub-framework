CleverBus integration framework
====================================================================

Goal: create base for next integration projects where common useful and often used functionality will be
encapsulated. Framework offers best-practice implementations from real integration projects.


Maven modules
----------------------------
- common: common and handy functionality useful for all other modules
- core: core of the framework, contains most of the functionality
- web-admin: admin web GUI
